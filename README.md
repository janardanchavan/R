## Population Mean
<img src="https://latex.codecogs.com/svg.latex?\mu=\frac{\displaystyle\sum_{i=1}^{N}x_i}{N}" />
N = number of items in the population

## Sample Mean
<img src="https://latex.codecogs.com/svg.latex?\mu=\frac{\displaystyle\sum_{i=1}^{n}x_i}{n}" />
n = number of items in the sample
